# Create New Branch

Follow the skill contract in `skills/new-branch.md`.

## Task

Create a new branch for: $ARGUMENTS

## Quick Reference

Branch types:
- `feature` — New functionality
- `fix` — Bug fix
- `refactor` — Code restructuring
- `chore` — Maintenance tasks

Format: `{type}/{description}` (e.g., `feature/add-prompt-versioning`)
