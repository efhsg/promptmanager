# Skill Routing

When working on specific file types or topics, proactively load the relevant skill.

## File Pattern Routing

| When editing | Load skill |
|--------------|------------|
| `yii/migrations/*` | `skills/migration.md` |
| `yii/tests/*Test.php` | `rules/testing.md` |
| `yii/controllers/*` | `skills/controller-action.md` |
| `yii/models/*` | `skills/model.md` |
| `yii/services/*` | `skills/service.md` |
| `yii/common/enums/*` | `skills/enum.md` |
| `yii/exceptions/*` | `skills/error-handling.md` |
| `yii/widgets/*` | `skills/frontend-design.md` |
| `yii/presenters/*` | `skills/service.md` |
| `yii/helpers/*` | `skills/service.md` |
| `yii/components/*` | `skills/service.md` |

## Topic Routing

| When discussing | Load skill |
|-----------------|------------|
| Quill, editor, Delta | `skills/frontend-design.md` |
| Placeholders, rich text | `skills/frontend-design.md` |
| Validation, form rules | `skills/form.md` |
| Search, filtering, listing | `skills/search.md` |
| Error handling, exceptions | `skills/error-handling.md` |
| External review, triage, review feedback | `skills/triage-review.md` |
